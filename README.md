# jan 天气

一款简单的天气小程序软件


#### taro app   
https://taro-docs.jd.com/taro/docs/GETTING-STARTED 

### 使用
``````
# yarn
$ yarn dev:weapp
$ yarn build:weapp
# npm script
$ npm run dev:weapp
$ npm run build:weapp
# 仅限全局安装
$ taro build --type weapp --watch
$ taro build --type weapp
# npx 用户也可以使用
$ npx taro build --type weapp --watch
$ npx taro build --type weapp
``````
